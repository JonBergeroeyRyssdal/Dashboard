## Project: Reception Management Dashboard for WeDeliverTECH™

### Project Description
This project involves creating a **Reception Management Dashboard** for **WeDeliverTECH™**, a fictional company. The dashboard allows employees and administrators to manage deliveries, track packages, and generate reports for an efficient reception process.

### Project Structure  

```
Main Project Folder/
│
├── Documentation/
│   └── Jira.pdf                # Contains the project's Jira documentation.
│
├── Web Application/
│   ├── Company logo.png        # The logo used in the web application.
│   ├── index.html              # The main HTML file for the application.
│   ├── style.css               # The CSS file for styling the application.
│   └── wdt_app.js              # The JavaScript file containing the application's logic.
│
├── README.md                   # The file you are currently reading.
```

---

### Explanation  

1. **Documentation/**: This folder contains all project-related documentation.  
   - `Jira.pdf`: Documentation generated using Jira, outlining tasks, planning, or workflow related to the project.  

2. **Web Application/**: This folder contains all files related to the implementation of the web application.  
   - `Company logo.png`: A placeholder logo or branding image for the company.  
   - `index.html`: The main entry point for the web application. Open this file in a browser to run the project.  
   - `style.css`: Stylesheet for customizing the appearance of the web application.  
   - `wdt_app.js`: JavaScript file containing the logic and interactivity for the web application.  

3. **README.md**: This file provides instructions and information about the project.  

---

### Technologies & Libraries Used  

- **Bootstrap 5**: For the layout and responsive design.  
  - Included via CDN.  
  - CSS: `<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha3/dist/css/bootstrap.min.css" rel="stylesheet">`  
  - JavaScript: `<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha3/dist/js/bootstrap.bundle.min.js"></script>`  

- **jQuery**: For handling DOM manipulation and event handling.  
  - Included via CDN.  
  - JavaScript: `<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>`  

- **Font Awesome**: For icons such as car and motorcycle.  
  - Included via CDN.  
  - CSS: `<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css" rel="stylesheet">`  

- **Random User API**: Used to generate demo staff data.  
  - API Endpoint: `https://randomuser.me/api/`  
  - No setup is required; fetch requests are made directly from the client-side.  

- **Version Control**: Git, GitHub.  

---

### Installation & Setup  

1. No backend is required. The system runs entirely on the client-side using HTML, CSS, and JavaScript.  
2. Ensure you have a stable internet connection to load external libraries via CDN.  
3. Open `index.html` in your browser to run the project.
